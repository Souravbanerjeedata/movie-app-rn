# 🎬 MovieApp RN

A modern, beautiful **React Native** movie discovery and streaming app built with **Expo** and **TypeScript**. Browse trending movies & TV series, search, view details, and enjoy a Netflix-like experience.

## ✨ Features

- **🔐 Authentication** — Login & Sign-up screens
- **🏠 Home Feed** — Trending, Popular, Top Rated movies & series
- **🎥 Movies & TV Series** — Dedicated tabs with filtering
- **🔍 Search** — Find movies and shows easily
- **📱 Movie Details** — Rich detail view with backdrop, overview, genres, similar recommendations
- **📚 Continue Watching** — Resume your favorite content
- **🎨 Modern UI** — Dark theme, smooth animations, responsive design
- **🌐 TMDB Integration** — Real-time data from The Movie Database API

## 🛠 Tech Stack

- **Framework**: React Native + Expo (Router)
- **Language**: TypeScript
- **Styling**: StyleSheet + Expo Linear Gradient
- **API**: Axios + TMDB API
- **Navigation**: Expo Router (File-based)
- **UI Components**: Custom reusable components (MovieCard, AppButton, etc.)
- **State Management**: React hooks + Custom `useFetch`

### Prerequisites
- Node.js (v18+)
- Expo CLI (`npm install -g expo-cli`)
- TMDB Account (for API key)
